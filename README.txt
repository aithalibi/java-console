Application To-Do List (Gestionnaire de Tâches en Console)
====================================================

Description
-----------
Cette application console en Java permet de gérer une liste de tâches personnelles. Elle offre les fonctionnalités suivantes :
- Ajouter des tâches
- Afficher toutes les tâches
- Supprimer des tâches
- Marquer des tâches comme terminées
- Sauvegarder automatiquement les tâches dans un fichier
- Charger les tâches au démarrage

Fonctionnalités
--------------
- Gestion complète des tâches (ajout, suppression, affichage)
- Sauvegarde automatique dans un fichier texte
- Marquage des tâches comme terminées
- Affichage séparé des tâches terminées et non terminées
- Tri des tâches par ordre alphabétique
- Horodatage des tâches

Structure du Projet
------------------
- Tache.java : Classe représentant une tâche
- GestionnaireTaches.java : Classe gérant la logique de l'application
- Application.java : Point d'entrée du programme

Utilisation
-----------
1. Compiler le projet : javac *.java
2. Exécuter l'application : java Application
3. Suivre les instructions du menu pour gérer vos tâches

Les tâches sont automatiquement sauvegardées dans le fichier "taches.txt"
